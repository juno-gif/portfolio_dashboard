# 📊 포트폴리오 대시보드

국내·미국 주식 보유 현황을 한눈에 확인할 수 있는 개인 포트폴리오 대시보드입니다.
CSV 파일 업로드만으로 실시간 현재가 조회, 계좌별·섹터별 손익 분석을 바로 볼 수 있습니다.

## 주요 기능

### 포트폴리오 탭

- **실시간 현재가 조회** — 국내 종목(네이버 금융), 미국 종목(Yahoo Finance), 환율(open.er-api.com) 자동 fetch
- **오늘 손익 / 전체 수익률** — 전일종가 대비 당일 손익 및 평균단가 대비 누적 수익률 계산
- **계좌별 요약 카드** — ISA, 연금저축, IRP, CMA 등 계좌 단위 평가금액 및 수익률 (카드 수에 따라 폭 자동 조절)
- **섹터 비중 파이차트** — 미국지수, 국내지수, 금, 방산/테마, 채권/혼합 등 섹터별 배분 시각화
- **종목별 테이블** — 컬럼 클릭으로 오름차순/내림차순 정렬
- **종목 드릴다운** — 종목 클릭 시 계좌별 보유 현황 드로어 표시
- **기타 자산 관리** — 펀드·금현물 등 자동 조회가 어려운 자산을 직접 입력해 총 평가금액에 합산
- **CSV 드래그 앤 드롭 / 재업로드** — 파일을 화면에 드래그하거나 버튼으로 교체 가능
- **CSV 다운로드** — 현재 데이터를 CSV로 내보내기

### 미래 예측 탭

- **50년 연도별 자산 추이** — 현재 자산 기준으로 연 수익률·현금흐름을 반영한 미래 자산 시뮬레이션
- **현금흐름 이벤트** — 일회성(퇴직금 등) 및 반복(국민연금, 은퇴 후 생활비 등) 이벤트를 자유롭게 추가
- **인플레이션 반영 현재가치** — 인플레이션율(%) 입력 시 매 연도 기말 자산을 현재 구매력 기준으로 환산해 표시
- **바 차트** — 연도별 기말 자산 추이, 이벤트 발생 연도는 별도 색상으로 표시
- **연도별 상세 테이블** — 기초 / 입출금 / 기말 / 증감 / 증감(월) / 현재가치

## 데이터 저장 구조

URL에 포함된 `?token=` 값을 기준으로 데이터를 격리하여 저장합니다.

| 데이터 | 저장 위치 |
|--------|-----------|
| 포트폴리오 CSV | Upstash Redis KV + localStorage |
| 기타 자산 | Upstash Redis KV + localStorage |
| 미래 예측 파라미터 | Upstash Redis KV + localStorage |

- **KV 우선**: 토큰 URL로 접속 시 KV에서 불러와 다른 기기에서도 동일한 데이터를 확인 가능
- **localStorage 동기화**: KV에 없는 데이터는 localStorage에서 불러오고 KV에 자동 업로드
- **URL 복사**: "내 포트폴리오 URL 복사" 버튼으로 현재 데이터 링크를 공유

## CSV 형식

```
계좌,종목명,종목번호,수량,평균단가,단위
ISA,KODEX 미국S&P500,379800,286,20285,KRW
CMA,AAPL,AAPL,8,257,USD
```

| 컬럼 | 설명 |
|------|------|
| 계좌 | 계좌 이름 (자유 입력) |
| 종목명 | 표시될 종목 이름 |
| 종목번호 | 국내 ETF/주식: 6자리 KRX 코드, 미국 주식: 티커 |
| 수량 | 보유 수량 |
| 평균단가 | 매입 평균단가 (KRW 또는 USD) |
| 단위 | `KRW` 또는 `USD` |

## 시작하기

```bash
npm install
npm run dev
```

[http://localhost:3000](http://localhost:3000) 접속 시 샘플 데이터로 대시보드가 바로 표시됩니다.

### 환경 변수 (Vercel 배포 시 필요)

```
UPSTASH_REDIS_REST_URL=...
UPSTASH_REDIS_REST_TOKEN=...
```

## 기술 스택

- **Next.js 15** (App Router) · **React 19** · **TypeScript**
- **TailwindCSS v4** · **shadcn/ui**
- **Recharts** — 차트 시각화
- **papaparse** — CSV 파싱
- **Upstash Redis** — 서버리스 KV 스토어 (크로스 디바이스 데이터 동기화)

## 데이터 소스

| 데이터 | 출처 |
|--------|------|
| 국내 종목 현재가 | 네이버 금융 polling API |
| 미국 종목 현재가 | Yahoo Finance API v8 |
| USD/KRW 환율 | open.er-api.com |

> 외부 API는 Next.js API Route를 통해 서버 사이드에서 호출합니다 (CORS 우회).
