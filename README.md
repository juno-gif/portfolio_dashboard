# 📊 포트폴리오 대시보드

국내·미국 주식 보유 현황을 한눈에 확인할 수 있는 개인 포트폴리오 대시보드입니다.
CSV 파일 업로드만으로 실시간 현재가 조회, 계좌별·섹터별 손익 분석을 바로 볼 수 있습니다.

## 주요 기능

- **실시간 현재가 조회** — 국내 종목(네이버 금융), 미국 종목(Yahoo Finance), 환율(open.er-api.com) 자동 fetch
- **오늘 손익 / 전체 수익률** — 전일종가 대비 당일 손익 및 평균단가 대비 누적 수익률 계산
- **계좌별 요약 카드** — ISA, 연금저축, IRP, CMA 등 계좌 단위 평가금액 및 수익률
- **섹터 비중 파이차트** — 미국지수, 국내지수, 금, 방산/테마, 채권/혼합 등 섹터별 배분 시각화
- **종목별 테이블** — 컬럼 클릭으로 오름차순/내림차순 정렬
- **종목 드릴다운** — 종목 클릭 시 계좌별 보유 현황 드로어 표시
- **CSV 다운로드 / 재업로드** — 현재 데이터 CSV로 내보내기, 새 CSV로 교체 가능

## CSV 형식

```
계좌,종목명,종목번호,수량,평균단가,단위
ISA,KODEX 미국S&P500,379800,286,20285,KRW
CMA,AAPL,AAPL,8,257,USD
```

| 컬럼 | 설명 |
|------|------|
| 계좌 | 계좌 이름 (자유 입력) |
| 종목명 | 표시될 종목 이름 |
| 종목번호 | 국내 ETF/주식: 6자리 KRX 코드, 미국 주식: 티커 |
| 수량 | 보유 수량 |
| 평균단가 | 매입 평균단가 (KRW 또는 USD) |
| 단위 | `KRW` 또는 `USD` |

## 시작하기

```bash
npm install
npm run dev
```

[http://localhost:3000](http://localhost:3000) 접속 시 샘플 데이터로 대시보드가 바로 표시됩니다.

## 기술 스택

- **Next.js 15** (App Router) · **React 19** · **TypeScript**
- **TailwindCSS v4** · **shadcn/ui**
- **Recharts** — 섹터 파이차트
- **papaparse** — CSV 파싱

## 데이터 소스

| 데이터 | 출처 |
|--------|------|
| 국내 종목 현재가 | 네이버 금융 polling API |
| 미국 종목 현재가 | Yahoo Finance API v8 |
| USD/KRW 환율 | open.er-api.com |

> 외부 API는 Next.js API Route를 통해 서버 사이드에서 호출합니다 (CORS 우회).
